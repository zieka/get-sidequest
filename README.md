## Install on macOS

1. Go to the [latest release](https://github.com/zieka/get-sidequest/releases/latest)
2. Download the `.dmg` file
3. Open the `.dmg` and drag **Sidequest** into your Applications folder

## Install on Linux

### Debian / Ubuntu (.deb)

1. Go to the [latest release](https://github.com/zieka/get-sidequest/releases/latest)
2. Download the `.deb` file
3. Install with `sudo dpkg -i Sidequest_*.deb`

### Fedora / RHEL (.rpm)

1. Go to the [latest release](https://github.com/zieka/get-sidequest/releases/latest)
2. Download the `.rpm` file
3. Install with `sudo rpm -i Sidequest-*.rpm`

### Any distro (AppImage)

1. Go to the [latest release](https://github.com/zieka/get-sidequest/releases/latest)
2. Download the `.AppImage` file
3. Make it executable and run: `chmod +x Sidequest_*.AppImage && ./Sidequest_*.AppImage`
