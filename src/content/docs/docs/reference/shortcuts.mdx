---
title: Shortcuts Cheat Sheet
description: Complete reference of all Sidequest keyboard shortcuts.
---

Every keyboard shortcut available in Sidequest, organized by category. All shortcuts use the **Cmd** key on macOS.

Press **Cmd+/** inside the app to open the built-in shortcuts overlay.

## Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+[ | Navigate to previous quest | Global |
| Cmd+] | Navigate to next quest | Global |
| Cmd+Shift+P | Open command palette | Global |

## Settings and Help

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+, | Open settings | Global |
| Cmd+/ | Show keyboard shortcuts | Global |
| Cmd+R | Reload window | Global |

## Panel Toggles

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+1 | Toggle Workspaces sidebar | Global |
| Cmd+2 | Toggle Quest Log | Global |
| Cmd+3 | Toggle Terminal panel | Global |
| Cmd+4 | Toggle Changes panel | Global |
| Cmd+5 | Toggle first visible extension (e.g., Live Stats) | Global |
| Cmd+6 | Toggle second visible extension (e.g., Toolkit) | Global |
| Cmd+7 | Toggle third visible extension (e.g., Council) | Global |

Extension shortcuts (Cmd+5 and above) map to extensions in the order they appear in the bottom toolbar. Reorder extensions in **Settings > Extensions** to change the mapping.

## Terminal Font Size

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+= | Increase terminal font size | Global |
| Cmd+- | Decrease terminal font size | Global |

## Editing (Menu Bar)

Standard macOS edit shortcuts available via the Edit menu:

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+Z | Undo | Text fields |
| Cmd+Shift+Z | Redo | Text fields |
| Cmd+X | Cut | Text fields |
| Cmd+C | Copy | Text fields |
| Cmd+V | Paste | Text fields |
| Cmd+A | Select All | Text fields |

## Window

| Shortcut | Action | Context |
|----------|--------|---------|
| Cmd+M | Minimize window | Global |
| Cmd+W | Close window | Global |
| Ctrl+Cmd+F | Toggle fullscreen | Global |

## Command Palette

These actions are available from within the command palette (Cmd+Shift+P):

| Action | How |
|--------|-----|
| Search quests | Type quest name or prompt text |
| Search files | Type **>** then a filename |
| Jump to quest | Select from results |
| Create quest | Choose "Create Quest" action |
| Open in editor | Choose "Open in Editor" action |
| Activate backlog quest | Choose "Activate" action |

## Bottom Toolbar Shortcut Discovery

Hold the **Cmd** key for approximately one second to reveal shortcut labels above each panel toggle button in the bottom toolbar. Release Cmd to hide them.
